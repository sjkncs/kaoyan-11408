## 二、六大题型 × 解题套路详解

### 题型A：数项级数敛散性判断

**核心判断流程（决策树）：**

```
拿到 ∑aₙ
  │
  ├─ aₙ ≥ 0（正项级数）？
  │    ├─ 含 n! 或 nⁿ → 比值判别法 (达朗贝尔)
  │    │    ρ = lim aₙ₊₁/aₙ：ρ<1收敛，ρ>1发散，ρ=1失效
  │    ├─ 含 n次根号 → 根值判别法 (柯西)
  │    │    ρ = lim ⁿ√aₙ：同上
  │    ├─ 多项式分式 → p级数比较法
  │    │    aₙ ~ C/nᵖ → p>1收敛, p≤1发散
  │    └─ 含 ln n 等慢增函数 → 比较判别法（与 1/nᵃ⁻ᵋ 比较）
  │
  ├─ aₙ 正负交替（交错级数）？
  │    └─ 莱布尼茨判别法：
  │         ① |aₙ| 单调递减（构造 f(x) 求导验证）
  │         ② lim aₙ = 0
  │         → 两个条件都满足则收敛
  │
  └─ 判断绝对 or 条件收敛？
       ├─ ∑|aₙ| 收敛 → 绝对收敛
       ├─ ∑|aₙ| 发散但 ∑aₙ 收敛 → 条件收敛
       └─ 常用技巧：有理化处理（乘共轭）
```

**典型例题与"为什么这么做"：**

**题6** — ∑(-1)ⁿ(√(n+1)-√n)/nᵖ 的敛散性
- **为什么用等价无穷小？** 因为 √(n+1)-√n 形式复杂，需化简才能判断阶数
- **关键步骤：** 乘共轭 → (√(n+1)-√n) = 1/(√(n+1)+√n) ~ 1/(2√n)
- **所以** aₙ ~ 1/(2n^(p+1/2))，用 p 级数比较法
- **结论：** p > 1/2 绝对收敛；-1/2 < p ≤ 1/2 条件收敛（莱布尼茨）；p ≤ -1/2 发散
- **理论基础：** 等价无穷小替换 + p级数判别法 + 莱布尼茨判别法

**题11（第二部分）** — ∑(e^(-n⁻ᵃ ln n) - 1) 的敛散性
- **为什么用等价无穷小？** e^u - 1 ~ u (u→0)，将指数型转化为代数型
- **关键步骤：** e^(-n⁻ᵃ ln n) - 1 ~ -n⁻ᵃ ln n = -ln n / nᵃ
- **再分类讨论：** a > 1 时收敛（取 ε 使 a-ε > 1）；0 < a ≤ 1 时发散
- **理论基础：** 等价无穷小 eᵘ-1 ~ u + 比较判别法

**固定套路总结：**
1. **看到根号差** → 乘共轭化简
2. **看到 e^u - 1** → 等价替换为 u
3. **看到 ln n / nᵃ** → 比较法，a>1 收敛，a≤1 发散
4. **验证莱布尼茨的单调性** → 构造连续函数 f(x)，用 f'(x) 的符号判断

### 题型B：幂级数收敛半径与收敛域

**核心公式：**

| 方法 | 公式 | 适用场景 |
|------|------|----------|
| 比值法 | R = lim\|aₙ/aₙ₊₁\| | 最通用，首选 |
| 根值法 | R = 1/lim ⁿ√\|aₙ\| | aₙ 含 n 次方时 |
| 变量替换 | 令 t = x-c 或 t = x² | 缺项幂级数、平移级数 |

**收敛域 = 收敛区间 + 端点单独判断**

**典型例题与"为什么这么做"：**

**题12** — ∑aₙ(x+2)ⁿ 在 x=-2 收敛，在 x=0 发散
- **为什么能推出 R？** x=-2 是展开中心，在此收敛说明 R≥0（trivial）
- **关键推理：** x=0 发散 → |0-(-2)|=2 ≥ R，又在 x=-2 收敛 → R>0
- **所以** R=1（因为 -3 < x < -1），端点 x=-√2 处 |(-√2)+2| = 2-√2 < 1，绝对收敛

**题14** — ∑(aₙ + bₙ/2ⁿ)xⁿ 的收敛半径
- **为什么拆分？** ∑aₙxⁿ 的 R₁=r，∑(bₙ/2ⁿ)xⁿ 需变量替换 y=x/2
- **关键步骤：** ∑bₙ(x/2)ⁿ 的收敛条件 |x/2| < r → |x| < 2r
- **所以** 两部分 R 不同（r vs 2r），合并后 R = min{r, 2r} = r
- **理论基础：** 幂级数逐项加减，R ≥ min{R₁, R₂}

**固定套路总结：**
1. **求 R** → 比值法（lim|aₙ/aₙ₊₁|）是万能钥匙
2. **端点 x=±R** → 必须代入后单独判断（p级数/交错级数）
3. **缺项级数**（只有 x²ⁿ）→ 令 t=x²，先求 t 的范围再开方
4. **系数含 2ⁿ 等** → 变量替换 y=kx，注意替换后 R 要反推

### 题型C：求和函数 S(x)

**这是本章最核心、考频最高的题型。** 基本策略是将未知级数转化为已知级数。

**两大操作方向（互逆）：**

```
∑ n·aₙ·xⁿ⁻¹  ←(求导)—  S(x)  —(积分)→  ∑ aₙ/(n+1) · xⁿ⁺¹
   ↓                                    ↓
  消去 n 的系数                      消去分母中的 n
```

**选择求导还是积分？看 aₙ 的结构：**

| aₙ 的特征 | 操作 | 目的 |
|-----------|------|------|
| 分子含 n, n² | 积分 | 消去分子的多项式因子 |
| 分母含 n, n(n+1) | 求导 | 消去分母的多项式因子 |
| 含 n! | 联想到 eˣ 展开 | 直接匹配 |
| 含 (-1)ⁿ/(2n+1) | 联想到 arctan x | 直接匹配 |
| 含 (-1)ⁿ/n | 联想到 ln(1+x) | 直接匹配 |

**典型例题与"为什么这么做"：**

**题15** — S(x) = ∑ xⁿ/((n+1)(n+2))
- **为什么先乘 x²？** 因为分母有 (n+1)(n+2)，需要两次求导才能消去
- **步骤链：** 
  1. 令 x²S(x) = ∑ xⁿ⁺²/((n+1)(n+2))
  2. 求导一次：[x²S(x)]' = ∑ xⁿ⁺¹/(n+1)
  3. 再求导：[x²S(x)]'' = ∑ xⁿ = 1/(1-x)
  4. 积分回去（两次），利用 S(0)=0 定常数
- **最终结果：** S(x) = (2-x)/(2x) + (1-x)ln(1-x)/x²，S(1)=1/2

**题9** — S(x) = ∑(n+1)(n+3)xⁿ
- **为什么用两次积分？** 分子含 n 的多项式（二次），积分两次化为几何级数
- **步骤链：**
  1. 拆为 h(x) = ∑(n+1)(n+2)xⁿ 和 g(x) = ∑(n+1)xⁿ
  2. h(x)：积分两次 → ∑xⁿ⁺² = x³/(1-x)，求导两次回去
  3. g(x)：积分一次 → ∑xⁿ⁺¹ = x²/(1-x)，求导一次回去
- **最终结果：** S(x) = 2/(1-x)³ + 1/(1-x)² - 3

**题21** — S(x) = ∑(1 + 1/n!)xⁿ
- **为什么拆分？** 1 + 1/n! 可以分成两个已知级数
- **步骤：** ∑xⁿ = x/(1-x)（几何级数）+ ∑xⁿ/n! = eˣ - 1
- **最终结果：** S(x) = x/(1-x) + eˣ - 1

**固定套路总结：**
1. **分母有 n 的乘积** → 逐项求导消分母（求导次数 = 分母中 n 的因子数）
2. **分子有 n 的乘积** → 逐项积分消分子（积分次数 = 分子中 n 的因子数）
3. **每次求导/积分后** → 检查是否变成几何级数 1/(1-x) 或 eˣ 等已知形式
4. **常数确定** → 代入 x=0，利用 S(0) = a₀ 或逐项验证
5. **端点处** → 若级数在端点收敛，利用连续性补定义

### 题型D：数值级数求和

**核心思想：数值级数 = 幂级数在某点的值。**

**常用已知展开式（必须背熟）：**

| 展开式 | 收敛域 | 常考代入值 |
|--------|--------|-----------|
| 1/(1-x) = ∑xⁿ | (-1,1) | x=1/2, -1/2 |
| eˣ = ∑xⁿ/n! | (-∞,+∞) | x=1, -1 |
| ln(1+x) = ∑(-1)ⁿ⁺¹xⁿ/n | (-1,1] | x=1 |
| arctan x = ∑(-1)ⁿx²ⁿ⁺¹/(2n+1) | [-1,1] | x=1 |
| sin x = ∑(-1)ⁿx²ⁿ⁺¹/(2n+1)! | (-∞,+∞) | x=π/2 |

**典型例题与"为什么这么做"：**

**题25** — ∑(-1)ⁿn²/2ⁿ
- **为什么构造 S(x) = ∑n²xⁿ？** 因为数值级数就是 S(-1/2)
- **两次积分消 n²：** ∫∫S(t)/t → ∑xⁿ = x/(1-x)，再求导两次回来
- **结果：** S(x) = x(1+x)/(1-x)³，代入 x=-1/2 得 -2/27

**题31** — ∑(-1)ⁿ aₙaₙ₊₂，其中 aₙ = ∫₀^∞ e^(-n²x²)dx
- **为什么先算积分？** aₙ 是含参积分，用替换 u=nx 得 aₙ = √π/(2n)
- **然后裂项：** 1/(n(n+2)) = (1/n - 1/(n+2))/2 → 交错级数裂项求和
- **结果：** -π/16

**固定套路总结：**
1. **看到 ∑(含n表达式)/kⁿ** → 构造幂级数，代入 x=1/k
2. **看到 n! 相关** → 联想 eˣ 展开，代入 x=1 或 -1
3. **看到积分定义的 aₙ** → 先计算积分化简，再处理级数
4. **裂项** → 1/(n(n+k)) = (1/k)(1/n - 1/(n+k))

### 题型E：傅里叶级数

**核心公式（数学一专属）：**

```
f(x) ~ a₀/2 + ∑(aₙcos(nx) + bₙsin(nx))

a₀ = (1/π)∫₋π^π f(x)dx
aₙ = (1/π)∫₋π^π f(x)cos(nx)dx
bₙ = (1/π)∫₋π^π f(x)sin(nx)dx
```

**狄利克雷收敛定理（核心工具）：**
- 连续点：S(x) = f(x)
- 间断点：S(x) = [f(x⁺) + f(x⁻)] / 2
- 端点 x=±π：S(±π) = [f(-π⁺) + f(π⁻)] / 2

**典型例题与"为什么这么做"：**

**题37** — 求 ∑aₙ（傅里叶余弦系数之和）
- **为什么用狄利克雷定理？** 因为 ∑aₙ 不是直接可求的，但 S(0) = a₀/2 + ∑aₙ
- **关键步骤：** S(0) = [f(0⁺) + f(0⁻)]/2 = 1/2
- **所以** ∑aₙ = S(0) - a₀/2 = 1/2 - (π/4 + 1/2) = -π/4
- **理论基础：** 狄利克雷定理将级数值与函数值建立联系

**题40** — 偶延拓后用傅里叶级数求 ∑(-1)ⁿ⁺¹/n²
- **为什么偶延拓？** 题目要求余弦级数 → 偶延拓 → bₙ=0
- **关键步骤：** 算出 aₙ = 4(-1)ⁿ⁺¹/n²，代入 x=0 得 S(0)=f(0)=1
- **所以** 1 = a₀/2 + ∑aₙ → 解出 ∑(-1)ⁿ⁺¹/n² = π²/12
- **理论基础：** 傅里叶级数是"已知函数反推级数之和"的桥梁

**固定套路总结：**
1. **求 ∑1/n² 或 ∑(-1)ⁿ/n² 等** → 构造合适的 f(x)，展开傅里叶级数，代入特殊点
2. **偶函数** → 只有余弦项（bₙ=0）；奇函数 → 只有正弦项（aₙ=0）
3. **间断点处** → S(x) 取左右极限的平均值（常考陷阱）
4. **周期延拓** → 注意延拓后的不连续点位置

### 题型F：跨章节综合题

**题7** — eˣ-e⁻ˣ 的中值定理 + 级数敛散性
- **为什么用拉格朗日中值定理？** 题目给出 e^(aₙ)-e^(-aₙ) = (e^(bₙ)+e^(-bₙ))aₙ
- **核心推理链：** 中值定理 → 建立 aₙ 与 bₙ 的不等关系 → 比较判别法
- **技巧：** 用积分比较证明 eˣ-e⁻ˣ > 2x，得到 aₙ/2 < bₙ < aₙ

**题10** — 递推数列 xₙ₊₁ = (a+xₙ)/(1+xₙ) 的级数
- **为什么构造 |xₙ₊₁-xₙ| ≤ k|xₙ-xₙ₋₁|？** 这是压缩映射的思想
- **关键：** k = (1-a)/(1+a) < 1 → |xₙ₊₁-xₙ| ≤ kⁿ⁻¹|x₂-x₁| → 几何级数控制
- **所以** ∑(xₙ₊₁-xₙ) 绝对收敛 → 部分和 Sₙ = xₙ₊₁-x₁ → 极限存在
- **理论基础：** 压缩映射原理 + 比较判别法

## 四、必须熟记的核心公式

### 常用等价无穷小（n→∞ 时）
```
√(n+1) - √n ~ 1/(2√n)
e^u - 1 ~ u (u→0)
ln(1+u) ~ u (u→0)
ln n / nᵃ → 0 (a>0, 但比 1/nᵃ⁻ᵋ 慢)
```

### 幂级数运算公式
```
逐项求导：(∑aₙxⁿ)' = ∑naₙxⁿ⁻¹       （收敛半径不变）
逐项积分：∫₀ˣ(∑aₙtⁿ)dt = ∑aₙxⁿ⁺¹/(n+1) （收敛半径不变）
柯西乘积：(∑aₙxⁿ)(∑bₙxⁿ) = ∑(∑ᵢ₌₀ⁿ aᵢbₙ₋ᵢ)xⁿ
```

### 经典级数之和
```
∑₁^∞ 1/n² = π²/6
∑₁^∞ (-1)ⁿ⁺¹/n² = π²/12
∑₁^∞ 1/(2n-1)² = π²/8
∑₀^∞ (-1)ⁿ/(2n+1) = π/4
```
