## 二、题型分类 × 解题套路详解

### 题型A：计算型行列式

**核心判断流程（决策树）：**

```
拿到 $n$ 阶行列式 $D_n$
  │
  ├─ 各行（列）元素结构相似，只差一个变量？
  │    └─ 加边法：增加一行一列，化为上/下三角
  │
  ├─ 按某行/列展开后，余子式与 $D_n$ 结构相同？
  │    └─ 递推法：建立 $D_n = f(D_{n-1})$ 的递推关系
  │
  └─ 能猜出 $D_n$ 的通项公式？
       └─ 数学归纳法：验证 $n=1$，假设 $n=k$ 成立，证 $n=k+1$
```

**典型例题与"为什么这么做"：**

**题1** — 计算 $n$ 阶行列式（加边法）

$$D_n = \begin{vmatrix} a_1 + x_1 & a_2 & \cdots & a_n \ a_1 & a_2 + x_2 & \cdots & a_n \ \vdots & \vdots & \ddots & \vdots \ a_1 & a_2 & \cdots & a_n + x_n \end{vmatrix}$$

- **为什么用加边法？** 每行都是 $(a_1, a_2, \ldots, a_n)$ 加上对角线的一个 $x_i$，结构高度对称
- **关键步骤：**
  1. 在最左边加一列 $(1, 0, 0, \ldots, 0)^T$，最上面加一行 $(1, a_1, a_2, \ldots, a_n)$
  2. 用 $r_i - r_1$（$i = 2, \ldots, n+1$）消去第一列的 $a_i$
  3. 化为下三角行列式，对角线元素为 $1, x_1, x_2, \ldots, x_n$
- **最终结果：** $D_n = \left(1 + \sum_{i=1}^n \frac{a_i}{x_i}\right) x_1 x_2 \cdots x_n$
- **理论基础：** 加边法不改变行列式的值（新增的行/列通过初等变换可消去）

**题2** — 递推法计算 $n$ 阶行列式

- **为什么用递推法？** 按第1列展开后，余子式 $M_{11}$ 与 $D_{n-1}$ 结构相同
- **关键步骤：**
  1. 按第1列展开：$D_n = b D_{n-1} + (-1)^{n+1} a_n \cdot (-1)^{n-1} = b D_{n-1} + a_n$
  2. 逐层展开递推：$D_n = b(bD_{n-2} + a_{n-1}) + a_n = b^2 D_{n-2} + a_{n-1}b + a_n$
  3. 最终展开到底：$D_n = b^{n-1}D_1 + a_2 b^{n-2} + \cdots + a_n$
- **最终结果：** $D_n = b^n + a_1 b^{n-1} + a_2 b^{n-2} + \cdots + a_{n-1}b + a_n$
- **理论基础：** 行列式按行（列）展开定理（Laplace 展开）
- **替代方法：** 数学归纳法——验证 $n=1$ 成立，假设 $n=k$ 成立，展开 $D_{k+1}$ 代入归纳假设

**固定套路总结：**

1. **加边法适用场景** → 行列式各行（列）除对角线外完全相同
2. **递推法适用场景** → 按某行/列展开后出现"缩小版"的同类行列式
3. **展开方向选择** → 选含零最多的行/列展开，减少计算量
4. **注意展开符号** → $(-1)^{i+j}$ 代数余子式的符号别漏

### 题型B：利用矩阵性质求行列式值

**这是强化篇的核心题型。** 不直接计算行列式，而是利用矩阵的相似、特征值等性质间接求值。

**核心知识链：**

$$A \sim B \implies |A| = |B|, \quad \text{特征值相同}, \quad \text{tr}(A) = \text{tr}(B)$$

$$A \text{ 的特征值为 } \lambda_1, \lambda_2, \ldots, \lambda_n \implies |f(A)| = f(\lambda_1) \cdot f(\lambda_2) \cdots f(\lambda_n)$$

**典型例题与"为什么这么做"：**

**题3** — $A \sim B$，求分块矩阵行列式 $\frac{64}{3}$

- **为什么用相似？** 题目给出 $A \sim B$，所以 $|A| = |B| = 2$，特征值相同
- **关键步骤：**
  1. $|(A+E)^{-1}| = \frac{1}{|A+E|}$，而 $A$ 的特征值为 $\lambda_i$，所以 $|A+E| = \prod(\lambda_i + 1) = 12$
  2. $|(2B)^*| = |2B|^{3-1} = (2^3 |B|)^2 = (8 \times 2)^2 = 256$
  3. 分块对角矩阵：$|(A+E)^{-1}| \cdot |(2B)^*| = \frac{1}{12} \times 256 = \frac{64}{3}$
- **理论基础：** 相似矩阵性质 + 伴随矩阵行列式 $|A^*| = |A|^{n-1}$ + 分块对角矩阵行列式

**题4** — $PA = AQ$，求 $|P - E|$，答案 (D)

- **为什么提取 $A$？** $PA = AQ \implies P = AQA^{-1}$（$A$ 可逆时）
- **关键步骤：**
  1. $P = AQA^{-1} \implies P - E = AQA^{-1} - E = A(Q - E)A^{-1}$
  2. $|P - E| = |A| \cdot |Q - E| \cdot |A^{-1}| = |Q - E|$
  3. $Q = \text{diag}(-1, -2, -3) \implies |Q - E| = (-2)(-3)(-4) = -24$
- **理论基础：** 相似变换保持行列式不变（$|PAP^{-1}| = |A|$）

**题5** — 求 $|f(A) - 6E|$，答案 $(-1)^n 10^r 9^{n-r}$

- **为什么用相似对角化？** $A$ 相似于某个分块对角阵，可以对角化后计算
- **关键步骤：**
  1. 利用 $P^{-1}AP = \begin{bmatrix} E_r & O \ O & O \end{bmatrix}$
  2. $f(A) - 6E$ 对角化后变为 $\begin{bmatrix} -10E_r & O \ O & -9E_{n-r} \end{bmatrix}$
  3. 行列式 $= (-10)^r \cdot (-9)^{n-r} = (-1)^n 10^r 9^{n-r}$
- **理论基础：** 若 $A = PBP^{-1}$，则 $f(A) = Pf(B)P^{-1}$

**题6** — 利用特征值求 $|3E - A|$，答案 (C) $= 8$

- **为什么先找特征值？** 题目给出 $|E-A| = 0, |E+A| = 0, |2E-A| = 0$
- **关键推理：**
  1. $|E - A| = 0 \implies \lambda_1 = 1$
  2. $|E + A| = 0 \implies |(-1)(E + A)| = 0 \implies |(-E) - A| = 0 \implies \lambda_2 = -1$
  3. $|2E - A| = 0 \implies \lambda_3 = 2$
  4. $3E - A$ 的特征值为 $3-1=2, 3-(-1)=4, 3-2=1$
  5. $|3E - A| = 2 \times 4 \times 1 = 8$
- **理论基础：** $|\lambda E - A| = 0 \iff \lambda$ 是 $A$ 的特征值

**固定套路总结：**

1. **看到 $A \sim B$** → 特征值相同、行列式相同、迹相同
2. **看到 $P^{-1}AP = B$** → $|f(A)| = |f(B)|$，对角阵时直接算对角元素乘积
3. **看到 $|kE - A| = 0$** → $k$ 是 $A$ 的特征值
4. **看到分块对角矩阵** → 行列式 = 各对角块行列式之积
5. **看到 $|A^*|$** → 用 $|A^*| = |A|^{n-1}$
6. **看到 $|(kA)^*|$** → $|kA|^{n-1} = (k^n|A|)^{n-1}$

### 题型C：综合题（线性无关 + 相似矩阵）

**题7** — 证明 $\alpha, A\alpha, A^2\alpha$ 线性无关 + 构造相似矩阵

**第(1)问：证明线性无关**

- **为什么构造 $k_1\alpha + k_2A\alpha + k_3A^2\alpha = 0$？** 这是线性无关的标准证法
- **关键步骤：**
  1. 设 $\alpha = c_1\xi_1 + c_2\xi_2 + c_3\xi_3$（用特征向量基展开，$c_i \neq 0$）
  2. $A\alpha = c_1\lambda_1\xi_1 + c_2\lambda_2\xi_2 + c_3\lambda_3\xi_3$
  3. $A^2\alpha = c_1\lambda_1^2\xi_1 + c_2\lambda_2^2\xi_2 + c_3\lambda_3^2\xi_3$
  4. 代入线性组合 $= 0$，因 $\xi_i$ 线性无关，得方程组：
     $$k_1 + k_2\lambda_i + k_3\lambda_i^2 = 0 \quad (i = 1, 2, 3)$$
  5. 系数矩阵是 Vandermonde 矩阵，$\lambda_i$ 互不相同 $\implies$ 行列式 $\neq 0 \implies k_1 = k_2 = k_3 = 0$
- **理论基础：** Vandermonde 行列式 $\prod_{i<j}(\lambda_j - \lambda_i) \neq 0$（$\lambda_i$ 互异时）

**第(2)问：构造相似矩阵 $B$**

- **为什么用 $A^3\alpha = A\alpha$？** 题目条件 $(A^3 - A)\alpha = 0$ 给出了 $A$ 作用在 $\alpha$ 上的递推关系
- **关键步骤：**
  1. $A^3\alpha = A\alpha$
  2. 令 $P = [\alpha, A\alpha, A^2\alpha]$（第(1)问已证可逆）
  3. $AP = [A\alpha, A^2\alpha, A^3\alpha] = [A\alpha, A^2\alpha, A\alpha]$
  4. 用 $P$ 的列表示 $AP$ 的每一列：
     - $A\alpha = 0 \cdot \alpha + 1 \cdot A\alpha + 0 \cdot A^2\alpha$
     - $A^2\alpha = 0 \cdot \alpha + 0 \cdot A\alpha + 1 \cdot A^2\alpha$
     - $A^3\alpha = A\alpha = 0 \cdot \alpha + 1 \cdot A\alpha + 0 \cdot A^2\alpha$
  5. 得 $AP = PB$，其中 $B = \begin{bmatrix} 0 & 0 & 0 \ 1 & 0 & 1 \ 0 & 1 & 0 \end{bmatrix}$
  6. $P^{-1}AP = B$，即 $A \sim B$
- **理论基础：** 若 $P$ 可逆且 $AP = PB$，则 $P^{-1}AP = B$

## 三、高频陷阱与易错点

| 陷阱 | 说明 | 出现题号 |
|------|------|----------|
| 展开符号搞错 | $(-1)^{i+j}$ 容易漏掉，特别是 $a_{n1}$ 位置 | 题2 |
| 加边法初等变换方向 | $r_i - r_1$ 还是 $c_j - c_1$，要统一 | 题1 |
| 相似 $\neq$ 合同 $\neq$ 等价 | 只有相似保证特征值相同 | 题3-6 |
| 伴随矩阵公式记错 | $|A^*| = |A|^{n-1}$ 不是 $|A|^n$ | 题3 |
| 特征值与 $|kE-A|$ 的符号 | $|E+A| = 0$ 意味着 $\lambda = -1$ 不是 $\lambda = 1$ | 题6 |
| Vandermonde 行列式公式 | $\prod_{1 \leq i < j \leq n}(\lambda_j - \lambda_i)$，下标别搞反 | 题7 |

## 四、必须熟记的核心公式

### 行列式基本性质
$$|AB| = |A| \cdot |B|$$
$$|kA| = k^n |A| \quad (A \text{ 为 } n \text{ 阶矩阵})$$
$$|A^T| = |A|$$
$$|A^{-1}| = \frac{1}{|A|}$$
$$|A^*| = |A|^{n-1}$$

### 分块对角矩阵
$$\begin{vmatrix} A & O \ O & B \end{vmatrix} = |A| \cdot |B|$$

### 相似矩阵的性质
$$A \sim B \implies |A| = |B|, \quad \text{tr}(A) = \text{tr}(B), \quad \text{特征值相同}$$

### 特征值与行列式
$$A \text{ 的特征值为 } \lambda_1, \ldots, \lambda_n \implies |A| = \lambda_1 \lambda_2 \cdots \lambda_n$$
$$|f(A)| = f(\lambda_1) \cdot f(\lambda_2) \cdots f(\lambda_n)$$

### Vandermonde 行列式
$$V_n = \begin{vmatrix} 1 & \lambda_1 & \lambda_1^2 & \cdots & \lambda_1^{n-1} \ 1 & \lambda_2 & \lambda_2^2 & \cdots & \lambda_2^{n-1} \ \vdots & \vdots & \vdots & \ddots & \vdots \ 1 & \lambda_n & \lambda_n^2 & \cdots & \lambda_n^{n-1} \end{vmatrix} = \prod_{1 \leq i < j \leq n} (\lambda_j - \lambda_i)$$
