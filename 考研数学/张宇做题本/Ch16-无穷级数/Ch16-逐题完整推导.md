## 题型A：敛散性判断

### 题6 — $\sum(-1)^n(\sqrt{n+1} - \sqrt{n})/n^p$ 的敛散性（按 $p$ 分类讨论）

**解析原文跳步：** "由于 $a_n \sim 1/(2n^{p+1/2})$"

**完整推导：**

**第1步：化简 $\sqrt{n+1} - \sqrt{n}$（乘共轭）**

为什么要乘共轭？因为 $\sqrt{n+1} - \sqrt{n}$ 直接看不出来是几阶无穷小。

```
\sqrt{n+1} - \sqrt{n} = \frac{[\sqrt{n+1} - \sqrt{n}] \cdot [\sqrt{n+1} + \sqrt{n}]}{\sqrt{n+1} + \sqrt{n}}
                      = \frac{(n+1) - n}{\sqrt{n+1} + \sqrt{n}}
                      = \frac{1}{\sqrt{n+1} + \sqrt{n}}
```

当 $n \to \infty$ 时，$\sqrt{n+1} \sim \sqrt{n}$，所以分母 $\sim \sqrt{n} + \sqrt{n} = 2\sqrt{n}$

```
\therefore \sqrt{n+1} - \sqrt{n} \sim \frac{1}{2\sqrt{n}} = \frac{1}{2n^{1/2}}
```

**第2步：代入原级数**

```
a_n = \frac{\sqrt{n+1} - \sqrt{n}}{n^p} \sim \frac{1/(2n^{1/2})}{n^p} = \frac{1}{2n^{p + 1/2}}
```

**第3步：按 $p+1/2$ 的值分类讨论**

用 $p$ 级数 $\sum 1/n^{p+1/2}$ 来判断：

| 情况 | $p$ 的范围 | $p+1/2$ | $\sum|a_n|$ | 结论 |
|------|----------|-------|-------|------|
| $p+1/2 > 1$ | $p > 1/2$ | $>1$ | 收敛 | 绝对收敛 |
| $p+1/2 \leq 1$ | $p \leq 1/2$ | $\leq 1$ | 发散 | 需进一步判断 |

**第4步：$p \leq 1/2$ 时，用莱布尼茨判交错级数**

需要验证两个条件：
- 条件1：$\lim a_n = 0$？ $\to$ 因为 $a_n \sim 1/(2n^{p+1/2})$，只要 $p+1/2 > 0$ 即 $p > -1/2$，就有 $a_n \to 0$ ✓
- 条件2：$a_n$ 单调递减？

验证单调性（构造连续函数）：
```
\text{令 } f(x) = \frac{\sqrt{x+1} - \sqrt{x}}{x^p} = \frac{1}{x^p(\sqrt{x+1} + \sqrt{x})}
```

分母 $g(x) = x^p(\sqrt{x+1} + \sqrt{x})$，当 $x$ 充分大时：
- $x^p$ 递增（$p > 0$ 时）或不变（$p = 0$）
- $\sqrt{x+1} + \sqrt{x}$ 递增

所以 $g(x)$ 递增 $\to$ $f(x) = 1/g(x)$ 递减 ✓

**结论汇总：**
- $p > 1/2$：绝对收敛
- $-1/2 < p \leq 1/2$：条件收敛（$|a_n|$ 发散但交错级数收敛）
- $p \leq -1/2$：发散（$a_n$ 不趋于 $0$，必要条件不满足）

> **对照模式手册：** 题型A 决策树 → 交错级数 → 莱布尼茨 → 单调性用 $f(x)$ 求导

### 题11 — $\sum(e^{-n^{-a} \cdot \ln n} - 1)$ 的敛散性

**解析原文跳步：** "$e^{-n^{-a} \ln n} - 1 \sim -n^{-a} \ln n$"

**完整推导：**

**第1步：识别结构 → 用等价无穷小**

当 $n \to \infty$ 时，$u = -n^{-a} \cdot \ln n = -(\ln n)/n^a$

因为 $a > 0$，所以 $n^a$ 增长比 $\ln n$ 快 $\to$ $u \to 0$

利用等价无穷小 $e^u - 1 \sim u$（当 $u \to 0$ 时）：

```
e^{-n^{-a} \ln n} - 1 \sim -n^{-a} \cdot \ln n = -\frac{\ln n}{n^a}
```

**第2步：判断 $\sum(\ln n)/n^a$ 的敛散性**

**解析跳步：** 直接说"$a>1$ 收敛，$a \leq 1$ 发散"，没说为什么。

原理：对任意 $\varepsilon > 0$，当 $n$ 充分大时 $\ln n < n^{\varepsilon}$

所以：
```
\frac{\ln n}{n^a} < \frac{n^{\varepsilon}}{n^a} = \frac{1}{n^{a-\varepsilon}}
```

- **$a > 1$ 时：** 取 $\varepsilon = (a-1)/2 > 0$，则 $a-\varepsilon = (a+1)/2 > 1$
  $\to$ $\sum 1/n^{(a+1)/2}$ 收敛（$p$ 级数，$p>1$）
  $\to$ 由比较法，$\sum(\ln n)/n^a$ 收敛

- **$0 < a \leq 1$ 时：** 因为 $\ln n > 1$（$n \geq 3$ 时）
  $\to$ $(\ln n)/n^a > 1/n^a$
  $\to$ 当 $a \leq 1$ 时，$\sum 1/n^a$ 发散（$p$ 级数，$p \leq 1$）
  $\to$ 由比较法，$\sum(\ln n)/n^a$ 发散

**结论：** $a > 1$ 收敛，$0 < a \leq 1$ 发散

> **对照模式手册：** 题型A 固定套路2 → 看到 $e^u-1$ → 等价替换为 $u$

## 题型B：收敛半径与收敛域

### 题15 — $S(x) = \sum x^n/((n+1)(n+2))$，求收敛域和 $S(x)$

**解析原文跳步：** "$R = \lim|a_n/a_{n+1}| = 1$" 和积分过程

**完整推导：**

**第1步：求收敛半径 $R$**

$a_n = 1/((n+1)(n+2))$

```
\left|\frac{a_n}{a_{n+1}}\right| = \frac{1/((n+1)(n+2))}{1/((n+2)(n+3))}
                                 = \frac{(n+2)(n+3)}{(n+1)(n+2)}
                                 = \frac{n+3}{n+1}
```

解析跳了中间过程，展开来看：
```
\frac{(n+2)(n+3)}{(n+1)(n+2)} = \frac{n+3}{n+1} \quad \leftarrow \text{上下都有 } (n+2)\text{，约掉}
                              = \frac{n+3}{n+1}
                              \to \frac{n}{n} = 1 \quad (n \to \infty)
```

所以 $R = 1$，收敛区间 $(-1, 1)$

**第2步：判断端点（解析跳了这步）**

$x = 1$ 时：$\sum 1/((n+1)(n+2)) \sim \sum 1/n^2$，$p=2 > 1$ $\to$ 收敛 ✓
$x = -1$ 时：$\sum (-1)^n/((n+1)(n+2))$，$|a_n| \sim 1/n^2$ 收敛 $\to$ 绝对收敛 ✓

**所以收敛域 = $[-1, 1]$**

**第3步：求 $S(x)$（$x \neq 0$ 时）**

解析原文："令 $[x^2 S(x)]' = \ldots$" 突然就出来了。这里逐步还原：

$S(x) = \sum_{n=1}^{\infty} x^n/((n+1)(n+2))$

分母有 $(n+1)(n+2)$，两个 $n$ 的因子 $\to$ 需要求导两次才能消掉。
但直接对 $S(x)$ 求导：$S'(x) = \sum n \cdot x^{n-1}/((n+1)(n+2))$，分母还在，不好处理。

**技巧：先凑出合适的幂次，让求导后刚好对齐。**

观察：如果先算 $x^2 S(x)$：
```
x^2 S(x) = \sum_{n=1}^{\infty} \frac{x^{n+2}}{(n+1)(n+2)}
```

对 $x^2 S(x)$ 求导：
```
[x^2 S(x)]' = \sum_{n=1}^{\infty} \frac{(n+2) \cdot x^{n+1}}{(n+1)(n+2)}
            = \sum_{n=1}^{\infty} \frac{x^{n+1}}{n+1} \quad \leftarrow (n+2) \text{ 约掉了！}
```

再求导一次：
```
[x^2 S(x)]'' = \sum_{n=1}^{\infty} \frac{(n+1) \cdot x^n}{n+1}
             = \sum_{n=1}^{\infty} x^n
             = x + x^2 + x^3 + \cdots
             = \frac{x}{1-x} \quad \leftarrow \text{几何级数！}
```

**第4步：积分回去（两次）**

第一次积分：
```
[x^2 S(x)]' = \int_0^x \frac{t}{1-t} \, dt
```

计算 $\int t/(1-t) \, dt$：
```
\frac{t}{1-t} = \frac{(t-1)+1}{1-t} = -1 + \frac{1}{1-t}
```

所以：
```
\int_0^x \left[-1 + \frac{1}{1-t}\right] dt = \left[-t - \ln(1-t)\right]_0^x = -x - \ln(1-x)
```

**第二次积分：**
```
x^2 S(x) = \int_0^x \left[-t - \ln(1-t)\right] dt
```

分开算：
```
\int_0^x (-t) \, dt = -\frac{x^2}{2}

\int_0^x (-\ln(1-t)) \, dt \text{：令 } u = 1-t,\; du = -dt
  = \int_1^{1-x} \ln(u) \cdot du \quad \leftarrow \text{注意上下限也要换}
  = \left[u \cdot \ln(u) - u\right]_1^{1-x}
  = (1-x)\ln(1-x) - (1-x) - (1 \cdot \ln 1 - 1)
  = (1-x)\ln(1-x) - 1 + x + 1
  = (1-x)\ln(1-x) + x
```

合起来：
```
x^2 S(x) = -\frac{x^2}{2} + (1-x)\ln(1-x) + x
```

**第5步：解出 $S(x)$**

```
S(x) = \frac{-x^2/2 + x + (1-x)\ln(1-x)}{x^2}
     = -\frac{1}{2} + \frac{1}{x} + \frac{(1-x)\ln(1-x)}{x^2}
     = \frac{2-x}{2x} + \frac{(1-x)\ln(1-x)}{x^2} \quad (x \neq 0,\; x \in [-1,1])
```

**第6步：补充 $x=0$ 和 $x=1$ 处的值**

$x=0$：$S(0) = a_1 = 1/(2 \cdot 3) + \cdots$ $\to$ 直接代入 $x=0$ 到级数：$S(0) = 0$（因为没有 $x^0$ 项，$n$ 从 $1$ 开始，$x=0$ 时所有项为 $0$）

$x=1$：$S(1) = \sum 1/((n+1)(n+2)) = \sum(1/(n+1) - 1/(n+2))$（裂项！）
```
= \left(\frac{1}{2} - \frac{1}{3}\right) + \left(\frac{1}{3} - \frac{1}{4}\right) + \left(\frac{1}{4} - \frac{1}{5}\right) + \cdots
= \frac{1}{2}
```

> **对照模式手册：** 题型C → 分母有 $n$ 的乘积 → 求导消分母 → 凑 $x^2$ 对齐幂次

## 题型C：求和函数 $S(x)$

### 题9 — $S(x) = \sum(n+1)(n+3)x^n$（$n$ 从 $1$ 开始）

**解析原文跳步：** 拆分为 $h(x)$ 和 $g(x)$ 后直接给结果

**完整推导：**

**第1步：拆分**

$(n+1)(n+3) = (n+1)(n+2) + (n+1)$

所以 $S(x) = h(x) + g(x)$，其中：
```
h(x) = \sum_{n=1}^{\infty} (n+1)(n+2)x^n
g(x) = \sum_{n=1}^{\infty} (n+1)x^n
```

**第2步：求 $g(x) = \sum(n+1)x^n$**

分子含 $(n+1)$ $\to$ 用积分消去。

```
\int_0^x g(t) \, dt = \int_0^x \left[\sum_{n=1}^{\infty} (n+1)t^n\right] dt
                    = \sum_{n=1}^{\infty} \frac{(n+1) \cdot x^{n+1}}{n+1}
                    = \sum_{n=1}^{\infty} x^{n+1}
                    = x^2 + x^3 + x^4 + \cdots
                    = \frac{x^2}{1-x}
```

求导回去：
```
g(x) = \left[\frac{x^2}{1-x}\right]'
     = \frac{2x(1-x) - x^2 \cdot (-1)}{(1-x)^2}
     = \frac{2x - 2x^2 + x^2}{(1-x)^2}
     = \frac{2x - x^2}{(1-x)^2}
```

解析直接写 $g(x) = (1-x)^{-2} - 1$，验证一下：
```
(1-x)^{-2} - 1 = \frac{1}{(1-x)^2} - 1 = \frac{1 - (1-x)^2}{(1-x)^2} = \frac{1-1+2x-x^2}{(1-x)^2} = \frac{2x-x^2}{(1-x)^2}
```
一样 ✓

**第3步：求 $h(x) = \sum(n+1)(n+2)x^n$**

分子含 $(n+1)(n+2)$，两个因子 $\to$ 积分两次。

第一次积分：
```
\int_0^x h(t) \, dt = \sum_{n=1}^{\infty} \frac{(n+1)(n+2) \cdot x^{n+1}}{n+1}
                    = \sum_{n=1}^{\infty} (n+2)x^{n+1}
```

令 $f(x) = \sum_{n=1}^{\infty} (n+2)x^{n+1}$

第二次积分：
```
\int_0^x f(t) \, dt = \sum_{n=1}^{\infty} \frac{(n+2) \cdot x^{n+2}}{n+2}
                    = \sum_{n=1}^{\infty} x^{n+2}
                    = x^3 + x^4 + x^5 + \cdots
                    = \frac{x^3}{1-x}
```

求导回去（第一次）：
```
f(x) = \left[\frac{x^3}{1-x}\right]'
     = \frac{3x^2(1-x) - x^3 \cdot (-1)}{(1-x)^2}
     = \frac{3x^2 - 3x^3 + x^3}{(1-x)^2}
     = \frac{3x^2 - 2x^3}{(1-x)^2}
```

再求导一次（第二次）：
```
h(x) = f'(x)
```

解析写的是 $h(x) = 2(1-x)^{-3} - 2$，我来验证：
```
2(1-x)^{-3} - 2 = \frac{2}{(1-x)^3} - 2 = \frac{2 - 2(1-x)^3}{(1-x)^3}
```

展开 $(1-x)^3 = 1 - 3x + 3x^2 - x^3$：
```
= \frac{2 - 2 + 6x - 6x^2 + 2x^3}{(1-x)^3}
= \frac{6x - 6x^2 + 2x^3}{(1-x)^3}
```

而我刚才算的 $f'(x) = [(3x^2 - 2x^3)/(1-x)^2]'$，用商法则：
```
= \frac{(6x - 6x^2)(1-x)^2 - (3x^2 - 2x^3) \cdot 2(1-x) \cdot (-1)}{(1-x)^4}
= \frac{(6x - 6x^2)(1-x) + 2(3x^2 - 2x^3)}{(1-x)^3}
= \frac{6x - 6x^2 - 6x^2 + 6x^3 + 6x^2 - 4x^3}{(1-x)^3}
= \frac{6x - 6x^2 + 2x^3}{(1-x)^3}
```

一样 ✓

**第4步：合并**

```
S(x) = h(x) + g(x) = \left[\frac{2}{(1-x)^3} - 2\right] + \left[\frac{1}{(1-x)^2} - 1\right]
     = \frac{2}{(1-x)^3} + \frac{1}{(1-x)^2} - 3
```

> **对照模式手册：** 题型C → 分子含 $n$ 的多项式 → 积分消分子

### 题21 — $S(x) = \sum_{n=1}^{\infty} a_n x^n$，其中 $a_n = 1 + 1/n!$

**解析原文跳步：** "$a_n - 1 = 1/(n!)$，迭代展开"

**完整推导：**

**第1步：理解递推关系**

题目给的是 $a_0 = 1$, $a_n - a_{n-1} = 1/(n \cdot (n-1)!)$ 对 $n \geq 1$\ldots
（实际上解析说的是：$a_n - 1 = (1/n)(a_{n-1} - 1)$）

让我重新理清：递推是 $a_n - 1 = (a_{n-1} - 1)/n$

设 $b_n = a_n - 1$，则 $b_n = b_{n-1}/n$

**第2步：迭代展开 $b_n$（解析跳了这步）**

```
b_n = \frac{b_{n-1}}{n}
    = \frac{b_{n-2}/(n-1)}{n} = \frac{b_{n-2}}{n(n-1)}
    = \frac{b_{n-3}}{n(n-1)(n-2)}
    \cdots
    = \frac{b_0}{n \cdot (n-1) \cdot (n-2) \cdots 2 \cdot 1}
    = \frac{b_0}{n!}
```

而 $b_0 = a_0 - 1 = 1 - 1 = 0$\ldots

等等，让我看 $a_0$：题中 $a_0 = 2$（因为 $a_n = 1 + 1/n!$，所以 $a_0 = 1 + 1/0! = 1 + 1 = 2$），$b_0 = 2 - 1 = 1$

所以 $b_n = 1/n!$，即 $a_n = 1 + 1/n!$

**第3步：拆分级数**

```
S(x) = \sum_{n=1}^{\infty} \left(1 + \frac{1}{n!}\right)x^n
     = \sum_{n=1}^{\infty} x^n + \sum_{n=1}^{\infty} \frac{x^n}{n!}
```

**第4步：分别求和**

第一个：$\sum_{n=1}^{\infty} x^n = x + x^2 + x^3 + \cdots = x/(1-x)$（$|x| < 1$）

第二个：$\sum_{n=1}^{\infty} x^n/n! = x/1! + x^2/2! + x^3/3! + \cdots = e^x - 1$（减去 $n=0$ 项）

**结果：** $S(x) = x/(1-x) + e^x - 1$

> **对照模式手册：** 题型C → 含 $n!$ → 联想 $e^x$ 展开 → 拆分

## 题型D：数值级数求和

### 题25 — $\sum_{n=1}^{\infty} (-1)^n \cdot n^2/2^n$

**解析原文跳步：** "构造 $S(x) = \sum n^2 x^n$" 然后直接给结果

**完整推导：**

**第1步：识别结构**

$(-1)^n \cdot n^2/2^n = n^2 \cdot (-1/2)^n$

所以这就是 $S(-1/2)$，其中 $S(x) = \sum_{n=1}^{\infty} n^2 x^n$

**第2步：求 $S(x) = \sum n^2 x^n$**

分子含 $n^2$ $\to$ 积分两次。

第一次积分：
```
\int_0^x \frac{S(t)}{t} \, dt = \int_0^x \left[\sum_{n=1}^{\infty} n^2 t^{n-1}\right] dt
                              = \sum_{n=1}^{\infty} n^2 \cdot \frac{x^n}{n}
                              = \sum_{n=1}^{\infty} n x^n
```

（为什么除以 $t$ 再积分？因为 $\int t^{n-1} dt = t^n/n$，刚好消去分子一个 $n$）

第二次积分：
```
\int_0^x \left[\frac{1}{u} \cdot \int_0^u \frac{S(t)}{t} \, dt\right] du = \int_0^x \left[\sum_{n=1}^{\infty} n u^{n-1}\right] du
                                                                        = \sum_{n=1}^{\infty} n \cdot \frac{x^n}{n}
                                                                        = \sum_{n=1}^{\infty} x^n
                                                                        = \frac{x}{1-x}
```

现在要反推回去。令 $G(x) = \int_0^x S(t)/t \, dt = \sum n x^n$

$G(x)$ 满足：$\int_0^x G(u)/u \, du = x/(1-x)$

两边求导：$G(x)/x = [x/(1-x)]' = [(1-x)-x(-1)]/(1-x)^2 = 1/(1-x)^2$

所以 $G(x) = x/(1-x)^2$

再求导：$S(x)/x = G'(x) = [x/(1-x)^2]'$

```
G'(x) = \frac{(1-x)^2 - x \cdot 2(1-x)(-1)}{(1-x)^4}
      = \frac{(1-x)^2 + 2x(1-x)}{(1-x)^4}
      = \frac{(1-x) + 2x}{(1-x)^3}
      = \frac{1+x}{(1-x)^3}
```

所以 $S(x) = x(1+x)/(1-x)^3$

**第3步：代入 $x = -1/2$**

```
S(-1/2) = \frac{(-1/2)(1 + (-1/2))}{(1 - (-1/2))^3}
        = \frac{(-1/2)(1/2)}{(3/2)^3}
        = \frac{-1/4}{27/8}
        = -\frac{1}{4} \times \frac{8}{27}
        = -\frac{2}{27}
```

> **对照模式手册：** 题型D → 看到 $\sum$ 含 $n/k^n$ → 构造幂级数代入 $x=-1/2$

### 题31 — $\sum_{n=1}^{\infty} (-1)^n a_n a_{n+2}$，其中 $a_n = \int_0^{\infty} e^{-n^2 x^2} dx$

**解析原文跳步：** "$a_n = \sqrt{\pi}/(2n)$" 和裂项求和

**完整推导：**

**第1步：计算 $a_n$**

```
a_n = \int_0^{\infty} e^{-n^2 x^2} \, dx
```

令 $u = nx$，则 $du = n \cdot dx$，$dx = du/n$

当 $x=0 \to u=0$；$x \to \infty \to u \to \infty$

```
a_n = \int_0^{\infty} e^{-u^2} \cdot \frac{du}{n}
    = \frac{1}{n} \cdot \int_0^{\infty} e^{-u^2} \, du
```

高斯积分 $\int_0^{\infty} e^{-u^2} du = \sqrt{\pi}/2$（这是必须记住的经典结果）

所以 $a_n = \sqrt{\pi}/(2n)$

**第2步：计算 $a_n \cdot a_{n+2}$**

```
a_n \cdot a_{n+2} = \frac{\sqrt{\pi}}{2n} \cdot \frac{\sqrt{\pi}}{2(n+2)}
                  = \frac{\pi}{4n(n+2)}
```

**第3步：裂项**

```
\frac{1}{n(n+2)} = \frac{1}{2} \cdot \left[\frac{1}{n} - \frac{1}{n+2}\right]
```

验证：$(1/2)[1/n - 1/(n+2)] = (1/2)[(n+2-n)/(n(n+2))] = (1/2)[2/(n(n+2))] = 1/(n(n+2))$ ✓

所以 $a_n \cdot a_{n+2} = (\pi/4) \cdot (1/2) \cdot [1/n - 1/(n+2)] = (\pi/8) \cdot [1/n - 1/(n+2)]$

**第4步：代入级数**

```
\sum_{n=1}^{\infty} (-1)^n a_n a_{n+2} = \frac{\pi}{8} \cdot \sum_{n=1}^{\infty} (-1)^n \left[\frac{1}{n} - \frac{1}{n+2}\right]
```

展开前几项看规律：
```
n=1:\; (-1)^1[1/1 - 1/3]     = -1 + 1/3
n=2:\; (-1)^2[1/2 - 1/4]     = +1/2 - 1/4
n=3:\; (-1)^3[1/3 - 1/5]     = -1/3 + 1/5
n=4:\; (-1)^4[1/4 - 1/6]     = +1/4 - 1/6
n=5:\; (-1)^5[1/5 - 1/7]     = -1/5 + 1/7
n=6:\; (-1)^6[1/6 - 1/8]     = +1/6 - 1/8
\cdots
```

按 $1/n$ 合并：
- $1/1$：只在 $n=1$ 出现 $\to$ $-1$
- $1/2$：只在 $n=2$ 出现 $\to$ $+1/2$
- $1/3$：$n=1$ 出现 $+1/3$，$n=3$ 出现 $-1/3$ $\to$ 抵消
- $1/4$：$n=2$ 出现 $-1/4$，$n=4$ 出现 $+1/4$ $\to$ 抵消
- $1/k$（$k \geq 3$）：$n=k-2$ 和 $n=k$ 各出现一次，符号相反 $\to$ 抵消

所以只剩 $-1 + 1/2 = -1/2$

```
\sum_{n=1}^{\infty} (-1)^n a_n a_{n+2} = \frac{\pi}{8} \cdot \left(-\frac{1}{2}\right) = -\frac{\pi}{16}
```

> **对照模式手册：** 题型D → 积分定义 $a_n$ → 先算积分 → 裂项

## 题型E：傅里叶级数

### 题40 — $f(x) = 1-x^2$ 在 $[0,\pi]$ 上，偶延拓后求 $\sum(-1)^{n+1}/n^2$

**解析原文跳步：** "$a_n = 4(-1)^{n+1}/n^2$" 的积分过程和最终代入

**完整推导：**

**第1步：偶延拓**

将 $f(x) = 1-x^2$ 从 $[0,\pi]$ 偶延拓到 $[-\pi,\pi]$：
```
F(x) = 1 - x^2, \quad x \in [-\pi, \pi]
```

偶函数 $\to$ $b_n = 0$，只有余弦级数。

**第2步：计算 $a_0$**

```
a_0 = \frac{1}{\pi} \int_{-\pi}^{\pi} F(x) \, dx
    = \frac{2}{\pi} \int_0^{\pi} (1-x^2) \, dx \quad \leftarrow \text{偶函数，积分区间折半}
    = \frac{2}{\pi} \left[x - \frac{x^3}{3}\right]_0^{\pi}
    = \frac{2}{\pi} \left[\pi - \frac{\pi^3}{3}\right]
    = 2 - \frac{2\pi^2}{3}
    = 2\left(1 - \frac{\pi^2}{3}\right)
```

**第3步：计算 $a_n$（$n \geq 1$）**

```
a_n = \frac{1}{\pi} \int_{-\pi}^{\pi} F(x)\cos(nx) \, dx
    = \frac{2}{\pi} \int_0^{\pi} (1-x^2)\cos(nx) \, dx \quad \leftarrow \text{偶函数} \times \text{偶函数} = \text{偶函数}
```

拆成两部分：
```
= \frac{2}{\pi} \left[\int_0^{\pi} \cos(nx) \, dx - \int_0^{\pi} x^2 \cos(nx) \, dx\right]
```

**第一个积分：**
```
\int_0^{\pi} \cos(nx) \, dx = \left[\frac{\sin(nx)}{n}\right]_0^{\pi} = \frac{\sin(n\pi)}{n} - 0 = 0 \quad (\text{因为 } \sin(n\pi) = 0)
```

**第二个积分（分部积分两次）：**
```
\int_0^{\pi} x^2 \cos(nx) \, dx
```

第一次分部积分：令 $u = x^2$, $dv = \cos(nx)\,dx$
```
du = 2x\,dx, \quad v = \frac{\sin(nx)}{n}

= \left[\frac{x^2 \sin(nx)}{n}\right]_0^{\pi} - \int_0^{\pi} \frac{2x \cdot \sin(nx)}{n} \, dx
= 0 - \frac{2}{n}\int_0^{\pi} x \cdot \sin(nx) \, dx
```

第二次分部积分：令 $u = x$, $dv = \sin(nx)\,dx$
```
du = dx, \quad v = -\frac{\cos(nx)}{n}

\int_0^{\pi} x \cdot \sin(nx) \, dx = \left[-\frac{x \cdot \cos(nx)}{n}\right]_0^{\pi} + \int_0^{\pi} \frac{\cos(nx)}{n} \, dx
                                    = \left[-\frac{\pi \cdot \cos(n\pi)}{n} + 0\right] + \left[\frac{\sin(nx)}{n^2}\right]_0^{\pi}
                                    = -\frac{\pi(-1)^n}{n} + 0
                                    = -\frac{\pi(-1)^n}{n}
```

代回去：
```
\int_0^{\pi} x^2 \cos(nx) \, dx = -\frac{2}{n} \cdot \left(-\frac{\pi(-1)^n}{n}\right) = \frac{2\pi(-1)^n}{n^2}
```

所以：
```
a_n = \frac{2}{\pi} \cdot \left[0 - \frac{2\pi(-1)^n}{n^2}\right] = -\frac{4(-1)^n}{n^2} = \frac{4(-1)^{n+1}}{n^2}
```

**第4步：傅里叶级数**

```
S(x) = \frac{a_0}{2} + \sum_{n=1}^{\infty} a_n \cos(nx)
     = \left(1 - \frac{\pi^2}{3}\right) + \sum_{n=1}^{\infty} \frac{4(-1)^{n+1}}{n^2} \cdot \cos(nx)
```

**第5步：代入 $x=0$ 求 $\sum(-1)^{n+1}/n^2$**

$x=0$ 是连续点，所以 $S(0) = F(0) = 1 - 0^2 = 1$

```
1 = \left(1 - \frac{\pi^2}{3}\right) + \sum_{n=1}^{\infty} \frac{4(-1)^{n+1}}{n^2} \cdot \cos(0)
1 = 1 - \frac{\pi^2}{3} + 4 \cdot \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n^2}

\frac{\pi^2}{3} = 4 \cdot \sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n^2}

\sum_{n=1}^{\infty} \frac{(-1)^{n+1}}{n^2} = \frac{\pi^2}{12}
```

> **对照模式手册：** 题型E → 构造 $f(x)$ → 傅里叶展开 → 代入特殊点反推级数和

## 题型F：综合题

### 题10 — $x_{n+1} = (a + x_n)/(1 + x_n)$（$0 < a < 1$, $x_1 > 0$），证 $\sum(x_{n+1} - x_n)$ 收敛

**解析原文跳步：** 压缩不等式的推导

**完整推导：**

**第1步：算 $x_{n+1} - x_n$**

```
x_{n+1} - x_n = \frac{a + x_n}{1 + x_n} - x_n
              = \frac{(a + x_n) - x_n(1 + x_n)}{1 + x_n}
              = \frac{a + x_n - x_n - x_n^2}{1 + x_n}
              = \frac{a - x_n^2}{1 + x_n}
```

**第2步：算相邻差的比（核心步骤）**

```
\frac{|x_{n+1} - x_n|}{|x_n - x_{n-1}|}
```

分子：
```
|x_{n+1} - x_n| = \left|\frac{a + x_n}{1 + x_n} - \frac{a + x_{n-1}}{1 + x_{n-1}}\right|
```

通分：
```
= \left|\frac{(a + x_n)(1 + x_{n-1}) - (a + x_{n-1})(1 + x_n)}{(1 + x_n)(1 + x_{n-1})}\right|
```

展开分子：
```
(a + x_n)(1 + x_{n-1}) = a + ax_{n-1} + x_n + x_n x_{n-1}
(a + x_{n-1})(1 + x_n) = a + ax_n + x_{n-1} + x_n x_{n-1}
```

相减：
```
= (ax_{n-1} + x_n) - (ax_n + x_{n-1})
= a(x_{n-1} - x_n) + (x_n - x_{n-1})
= (1 - a)(x_n - x_{n-1})
= (a - 1)(x_{n-1} - x_n)
```

所以：
```
|x_{n+1} - x_n| = \frac{|a - 1| \cdot |x_{n-1} - x_n|}{(1 + x_n)(1 + x_{n-1})}
                = \frac{(1 - a) \cdot |x_n - x_{n-1}|}{(1 + x_n)(1 + x_{n-1})}
```

**第3步：证明 $x_n \geq \sqrt{a}$**

用数学归纳法。假设 $x_n \geq 0$（由 $x_1 > 0$ 和递推保证）

```
x_{n+1} = \frac{a + x_n}{1 + x_n}
```

要证 $x_{n+1} \geq \sqrt{a}$：
```
x_{n+1} - \sqrt{a} = \frac{a + x_n}{1 + x_n} - \sqrt{a}
                   = \frac{a + x_n - \sqrt{a}(1 + x_n)}{1 + x_n}
                   = \frac{a - \sqrt{a} + x_n(1 - \sqrt{a})}{1 + x_n}
                   = \frac{\sqrt{a}(\sqrt{a} - 1) + x_n(1 - \sqrt{a})}{1 + x_n}
                   = \frac{(1 - \sqrt{a})(x_n - \sqrt{a})}{1 + x_n}
```

因为 $0 < a < 1$，所以 $1 - \sqrt{a} > 0$。如果 $x_n \geq \sqrt{a}$，则 $x_{n+1} - \sqrt{a} \geq 0$ ✓

**第4步：得出压缩比**

因为 $x_n \geq \sqrt{a}$，所以 $1 + x_n \geq 1 + \sqrt{a}$

```
|x_{n+1} - x_n| \leq \frac{1 - a}{(1 + \sqrt{a})^2} \cdot |x_n - x_{n-1}|
```

令 $k = (1-a)/(1+\sqrt{a})^2 < 1$

等等，解析用的是 $k = (1-a)/(1+a)$（如果 $x_n \geq a$ 而不是 $\sqrt{a}$）。让我重新看：

因为 $x_n \geq \sqrt{a} > a$（$0 < a < 1$ 时 $\sqrt{a} > a$），所以 $1 + x_n \geq 1 + a$

```
|x_{n+1} - x_n| \leq \frac{1-a}{(1+a)(1+a)} \cdot |x_n - x_{n-1}|
                \leq \frac{1-a}{(1+a)^2} \cdot |x_n - x_{n-1}|
```

实际上解析用了更简单的界：因为 $(1+x_n)(1+x_{n-1}) \geq (1+a)^2$\ldots 不，解析用的是 $k = (1-a)/(1+a)$

让我简化：只要证明 $k < 1$ 就行。
```
\frac{1-a}{1+a} < 1 \iff 1-a < 1+a \iff -a < a \iff a > 0 \quad \checkmark
```

**第5步：用几何级数控制**

```
|x_{n+1} - x_n| \leq k|x_n - x_{n-1}| \leq k^2|x_{n-1} - x_{n-2}| \leq \cdots \leq k^{n-1}|x_2 - x_1|
```

因为 $0 < k < 1$，$\sum k^{n-1}|x_2 - x_1|$ 是收敛的几何级数。

由比较判别法，$\sum|x_{n+1} - x_n|$ 收敛 $\to$ $\sum(x_{n+1} - x_n)$ 绝对收敛 ✓

**第6步：求级数之和**

部分和 $S_n = \sum_{k=1}^{n} (x_{k+1} - x_k) = x_{n+1} - x_1$（裂项！）

设 $\lim x_n = A$，代入递推：
```
A = \frac{a + A}{1 + A}
A(1 + A) = a + A
A + A^2 = a + A
A^2 = a
A = \sqrt{a}
```

所以 $\sum_{n=1}^{\infty} (x_{n+1} - x_n) = \lim(x_{n+1} - x_1) = \sqrt{a} - x_1$

> **对照模式手册：** 题型F → 压缩映射 → 几何级数控制 → 裂项求和

## 总结：解析中最常跳的5种步骤

| 跳步类型 | 解析写法 | 实际过程 | 出现频率 |
|----------|----------|----------|----------|
| 通分化简 | "经计算得..." | 展开分子、约分、合并 | 极高 |
| 分部积分 | "积分得..." | 令 $u$/$dv$、代入、可能需要两次 | 高 |
| 裂项抵消 | "$\sum=\ldots$" | 展开前几项，找抵消规律 | 高 |
| 求导公式 | "$[f(x)]'=\ldots$" | 用商法则/链式法则，逐步展开 | 中 |
| 端点判断 | "收敛域为..." | 代入 $x=\pm R$，用 $p$ 级数/莱布尼茨 | 高 |

以后看到解析里任何一句"显然""易知""经计算"，都是上面这5种之一，手动展开一遍就好。
