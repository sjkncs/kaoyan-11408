## 题型A：敛散性判断

### 题6 — ∑(-1)ⁿ(√(n+1) - √n)/nᵖ 的敛散性（按 p 分类讨论）

**解析原文跳步：** "由于 aₙ ~ 1/(2n^(p+1/2))"

**完整推导：**

**第1步：化简 √(n+1) - √n（乘共轭）**

为什么要乘共轭？因为 √(n+1) - √n 直接看不出来是几阶无穷小。

```
√(n+1) - √n = [√(n+1) - √n] · [√(n+1) + √n] / [√(n+1) + √n]
            = [(n+1) - n] / [√(n+1) + √n]
            = 1 / [√(n+1) + √n]
```

当 n→∞ 时，√(n+1) ~ √n，所以分母 ~ √n + √n = 2√n

```
∴ √(n+1) - √n ~ 1/(2√n) = 1/(2n^(1/2))
```

**第2步：代入原级数**

```
aₙ = (√(n+1) - √n) / nᵖ ~ [1/(2n^(1/2))] / nᵖ = 1 / (2n^(p + 1/2))
```

**第3步：按 p+1/2 的值分类讨论**

用 p 级数 ∑1/n^(p+1/2) 来判断：

| 情况 | p 的范围 | p+1/2 | ∑|aₙ| | 结论 |
|------|----------|-------|-------|------|
| p+1/2 > 1 | p > 1/2 | >1 | 收敛 | 绝对收敛 |
| p+1/2 ≤ 1 | p ≤ 1/2 | ≤1 | 发散 | 需进一步判断 |

**第4步：p ≤ 1/2 时，用莱布尼茨判交错级数**

需要验证两个条件：
- 条件1：lim aₙ = 0？ → 因为 aₙ ~ 1/(2n^(p+1/2))，只要 p+1/2 > 0 即 p > -1/2，就有 aₙ→0 ✓
- 条件2：aₙ 单调递减？

验证单调性（构造连续函数）：
```
令 f(x) = (√(x+1) - √x) / xᵖ = 1 / [xᵖ(√(x+1) + √x)]
```

分母 g(x) = xᵖ(√(x+1) + √x)，当 x 充分大时：
- xᵖ 递增（p > 0 时）或不变（p = 0）
- √(x+1) + √x 递增

所以 g(x) 递增 → f(x) = 1/g(x) 递减 ✓

**结论汇总：**
- p > 1/2：绝对收敛
- -1/2 < p ≤ 1/2：条件收敛（|aₙ| 发散但交错级数收敛）
- p ≤ -1/2：发散（aₙ 不趋于 0，必要条件不满足）

> **对照模式手册：** 题型A 决策树 → 交错级数 → 莱布尼茨 → 单调性用 f(x) 求导

### 题11 — ∑(e^(-n⁻ᵃ · ln n) - 1) 的敛散性

**解析原文跳步：** "e^(-n⁻ᵃ ln n) - 1 ~ -n⁻ᵃ ln n"

**完整推导：**

**第1步：识别结构 → 用等价无穷小**

当 n→∞ 时，u = -n⁻ᵃ · ln n = -(ln n)/nᵃ

因为 a > 0，所以 nᵃ 增长比 ln n 快 → u → 0

利用等价无穷小 eᵘ - 1 ~ u（当 u→0 时）：

```
e^(-n⁻ᵃ ln n) - 1 ~ -n⁻ᵃ · ln n = -(ln n) / nᵃ
```

**第2步：判断 ∑(ln n)/nᵃ 的敛散性**

**解析跳步：** 直接说"a>1 收敛，a≤1 发散"，没说为什么。

原理：对任意 ε > 0，当 n 充分大时 ln n < nᵋ

所以：
```
(ln n) / nᵃ < nᵋ / nᵃ = 1 / nᵃ⁻ᵋ
```

- **a > 1 时：** 取 ε = (a-1)/2 > 0，则 a-ε = (a+1)/2 > 1
  → ∑1/n^((a+1)/2) 收敛（p级数，p>1）
  → 由比较法，∑(ln n)/nᵃ 收敛

- **0 < a ≤ 1 时：** 因为 ln n > 1（n ≥ 3 时）
  → (ln n)/nᵃ > 1/nᵃ
  → 当 a ≤ 1 时，∑1/nᵃ 发散（p级数，p≤1）
  → 由比较法，∑(ln n)/nᵃ 发散

**结论：** a > 1 收敛，0 < a ≤ 1 发散

> **对照模式手册：** 题型A 固定套路2 → 看到 eᵘ-1 → 等价替换为 u

## 题型B：收敛半径与收敛域

### 题15 — S(x) = ∑ xⁿ/((n+1)(n+2))，求收敛域和 S(x)

**解析原文跳步：** "R = lim|aₙ/aₙ₊₁| = 1" 和积分过程

**完整推导：**

**第1步：求收敛半径 R**

aₙ = 1/((n+1)(n+2))

```
|aₙ/aₙ₊₁| = [1/((n+1)(n+2))] / [1/((n+2)(n+3))]
           = [(n+2)(n+3)] / [(n+1)(n+2)]
           = (n+3)/(n+1)
```

解析跳了中间过程，展开来看：
```
(n+2)(n+3) / (n+1)(n+2) = (n+3)/(n+1)   ← 上下都有 (n+2)，约掉
                         = (n+3)/(n+1)
                         → n/n = 1（n→∞）
```

所以 R = 1，收敛区间 (-1, 1)

**第2步：判断端点（解析跳了这步）**

x = 1 时：∑ 1/((n+1)(n+2)) ~ ∑ 1/n²，p=2 > 1 → 收敛 ✓
x = -1 时：∑ (-1)ⁿ/((n+1)(n+2))，|aₙ| ~ 1/n² 收敛 → 绝对收敛 ✓

**所以收敛域 = [-1, 1]**

**第3步：求 S(x)（x ≠ 0 时）**

解析原文："令 [x²S(x)]' = ... " 突然就出来了。这里逐步还原：

S(x) = ∑₁^∞ xⁿ/((n+1)(n+2))

分母有 (n+1)(n+2)，两个 n 的因子 → 需要求导两次才能消掉。
但直接对 S(x) 求导：S'(x) = ∑ n·xⁿ⁻¹/((n+1)(n+2))，分母还在，不好处理。

**技巧：先凑出合适的幂次，让求导后刚好对齐。**

观察：如果先算 x²S(x)：
```
x²S(x) = ∑₁^∞ xⁿ⁺²/((n+1)(n+2))
```

对 x²S(x) 求导：
```
[x²S(x)]' = ∑₁^∞ (n+2)·xⁿ⁺¹/((n+1)(n+2))
          = ∑₁^∞ xⁿ⁺¹/(n+1)           ← (n+2) 约掉了！
```

再求导一次：
```
[x²S(x)]'' = ∑₁^∞ (n+1)·xⁿ/(n+1)
           = ∑₁^∞ xⁿ
           = x + x² + x³ + ...
           = x/(1-x)                   ← 几何级数！
```

**第4步：积分回去（两次）**

第一次积分：
```
[x²S(x)]' = ∫₀ˣ t/(1-t) dt
```

计算 ∫t/(1-t) dt：
```
t/(1-t) = [(t-1)+1]/(1-t) = -1 + 1/(1-t)
```

所以：
```
∫₀ˣ [-1 + 1/(1-t)] dt = [-t - ln(1-t)]₀ˣ = -x - ln(1-x)
```

**第二次积分：**
```
x²S(x) = ∫₀ˣ [-t - ln(1-t)] dt
```

分开算：
```
∫₀ˣ (-t) dt = -x²/2

∫₀ˣ (-ln(1-t)) dt：令 u = 1-t, du = -dt
  = ∫₁^(1-x) ln(u) · du          ← 注意上下限也要换
  = [u·ln(u) - u]₁^(1-x)
  = (1-x)ln(1-x) - (1-x) - (1·ln1 - 1)
  = (1-x)ln(1-x) - 1 + x + 1
  = (1-x)ln(1-x) + x
```

合起来：
```
x²S(x) = -x²/2 + (1-x)ln(1-x) + x
```

**第5步：解出 S(x)**

```
S(x) = [-x²/2 + x + (1-x)ln(1-x)] / x²
     = -1/2 + 1/x + (1-x)ln(1-x)/x²
     = (2-x)/(2x) + (1-x)ln(1-x)/x²       （x ≠ 0, x ∈ [-1,1]）
```

**第6步：补充 x=0 和 x=1 处的值**

x=0：S(0) = a₁ = 1/(2·3) + ... → 直接代入 x=0 到级数：S(0) = 0（因为没有 x⁰ 项，n 从 1 开始，x=0 时所有项为 0）

x=1：S(1) = ∑1/((n+1)(n+2)) = ∑(1/(n+1) - 1/(n+2))（裂项！）
```
= (1/2 - 1/3) + (1/3 - 1/4) + (1/4 - 1/5) + ...
= 1/2
```

> **对照模式手册：** 题型C → 分母有 n 的乘积 → 求导消分母 → 凑 x² 对齐幂次

## 题型C：求和函数 S(x)

### 题9 — S(x) = ∑(n+1)(n+3)xⁿ（n 从 1 开始）

**解析原文跳步：** 拆分为 h(x) 和 g(x) 后直接给结果

**完整推导：**

**第1步：拆分**

(n+1)(n+3) = (n+1)(n+2) + (n+1)

所以 S(x) = h(x) + g(x)，其中：
```
h(x) = ∑₁^∞ (n+1)(n+2)xⁿ
g(x) = ∑₁^∞ (n+1)xⁿ
```

**第2步：求 g(x) = ∑(n+1)xⁿ**

分子含 (n+1) → 用积分消去。

```
∫₀ˣ g(t) dt = ∫₀ˣ [∑₁^∞ (n+1)tⁿ] dt
            = ∑₁^∞ (n+1)·xⁿ⁺¹/(n+1)
            = ∑₁^∞ xⁿ⁺¹
            = x² + x³ + x⁴ + ...
            = x²/(1-x)
```

求导回去：
```
g(x) = [x²/(1-x)]'
     = [2x(1-x) - x²·(-1)] / (1-x)²
     = [2x - 2x² + x²] / (1-x)²
     = (2x - x²) / (1-x)²
```

解析直接写 g(x) = (1-x)⁻² - 1，验证一下：
```
(1-x)⁻² - 1 = 1/(1-x)² - 1 = [1 - (1-x)²]/(1-x)² = [1-1+2x-x²]/(1-x)² = (2x-x²)/(1-x)²
```
一样 ✓

**第3步：求 h(x) = ∑(n+1)(n+2)xⁿ**

分子含 (n+1)(n+2)，两个因子 → 积分两次。

第一次积分：
```
∫₀ˣ h(t) dt = ∑₁^∞ (n+1)(n+2)·xⁿ⁺¹/(n+1)
            = ∑₁^∞ (n+2)xⁿ⁺¹
```

令 f(x) = ∑₁^∞ (n+2)xⁿ⁺¹

第二次积分：
```
∫₀ˣ f(t) dt = ∑₁^∞ (n+2)·xⁿ⁺²/(n+2)
            = ∑₁^∞ xⁿ⁺²
            = x³ + x⁴ + x⁵ + ...
            = x³/(1-x)
```

求导回去（第一次）：
```
f(x) = [x³/(1-x)]'
     = [3x²(1-x) - x³·(-1)] / (1-x)²
     = [3x² - 3x³ + x³] / (1-x)²
     = (3x² - 2x³) / (1-x)²
```

再求导一次（第二次）：
```
h(x) = f'(x)
```

解析写的是 h(x) = 2(1-x)⁻³ - 2，我来验证：
```
2(1-x)⁻³ - 2 = 2/(1-x)³ - 2 = [2 - 2(1-x)³]/(1-x)³
```

展开 (1-x)³ = 1 - 3x + 3x² - x³：
```
= [2 - 2 + 6x - 6x² + 2x³]/(1-x)³
= (6x - 6x² + 2x³)/(1-x)³
```

而我刚才算的 f'(x) = [(3x² - 2x³)/(1-x)²]'，用商法则：
```
= [(6x - 6x²)(1-x)² - (3x² - 2x³)·2(1-x)·(-1)] / (1-x)⁴
= [(6x - 6x²)(1-x) + 2(3x² - 2x³)] / (1-x)³
= [6x - 6x² - 6x² + 6x³ + 6x² - 4x³] / (1-x)³
= [6x - 6x² + 2x³] / (1-x)³
```

一样 ✓

**第4步：合并**

```
S(x) = h(x) + g(x) = [2/(1-x)³ - 2] + [1/(1-x)² - 1]
     = 2/(1-x)³ + 1/(1-x)² - 3
```

> **对照模式手册：** 题型C → 分子含 n 的多项式 → 积分消分子

### 题21 — S(x) = ∑₁^∞ aₙxⁿ，其中 aₙ = 1 + 1/n!

**解析原文跳步：** "aₙ - 1 = 1/(n!)，迭代展开"

**完整推导：**

**第1步：理解递推关系**

题目给的是 a₀ = 1, aₙ - aₙ₋₁ = 1/(n·(n-1)!) 对 n ≥ 1... 
（实际上解析说的是：aₙ - 1 = (1/n)(aₙ₋₁ - 1)）

让我重新理清：递推是 aₙ - 1 = (aₙ₋₁ - 1)/n

设 bₙ = aₙ - 1，则 bₙ = bₙ₋₁/n

**第2步：迭代展开 bₙ（解析跳了这步）**

```
bₙ = bₙ₋₁ / n
   = (bₙ₋₂ / (n-1)) / n = bₙ₋₂ / (n(n-1))
   = bₙ₋₃ / (n(n-1)(n-2))
   ...
   = b₀ / (n·(n-1)·(n-2)·...·2·1)
   = b₀ / n!
```

而 b₀ = a₀ - 1 = 1 - 1 = 0... 

等等，让我看 a₀：题中 a₀ = 2（因为 aₙ = 1 + 1/n!，所以 a₀ = 1 + 1/0! = 1 + 1 = 2），b₀ = 2 - 1 = 1

所以 bₙ = 1/n!，即 aₙ = 1 + 1/n!

**第3步：拆分级数**

```
S(x) = ∑₁^∞ (1 + 1/n!)xⁿ
     = ∑₁^∞ xⁿ + ∑₁^∞ xⁿ/n!
```

**第4步：分别求和**

第一个：∑₁^∞ xⁿ = x + x² + x³ + ... = x/(1-x)（|x| < 1）

第二个：∑₁^∞ xⁿ/n! = x/1! + x²/2! + x³/3! + ... = eˣ - 1（减去 n=0 项）

**结果：** S(x) = x/(1-x) + eˣ - 1

> **对照模式手册：** 题型C → 含 n! → 联想 eˣ 展开 → 拆分

## 题型D：数值级数求和

### 题25 — ∑₁^∞ (-1)ⁿ·n²/2ⁿ

**解析原文跳步：** "构造 S(x) = ∑n²xⁿ" 然后直接给结果

**完整推导：**

**第1步：识别结构**

(-1)ⁿ·n²/2ⁿ = n²·(-1/2)ⁿ

所以这就是 S(-1/2)，其中 S(x) = ∑₁^∞ n²xⁿ

**第2步：求 S(x) = ∑n²xⁿ**

分子含 n² → 积分两次。

第一次积分：
```
∫₀ˣ S(t)/t dt = ∫₀ˣ [∑₁^∞ n²tⁿ⁻¹] dt
              = ∑₁^∞ n² · xⁿ/n
              = ∑₁^∞ nxⁿ
```

（为什么除以 t 再积分？因为 ∫tⁿ⁻¹ dt = tⁿ/n，刚好消去分子一个 n）

第二次积分：
```
∫₀ˣ [1/u · ∫₀ᵘ S(t)/t dt] du = ∫₀ˣ [∑₁^∞ nuⁿ⁻¹] du
                              = ∑₁^∞ n · xⁿ/n
                              = ∑₁^∞ xⁿ
                              = x/(1-x)
```

现在要反推回去。令 G(x) = ∫₀ˣ S(t)/t dt = ∑nxⁿ

G(x) 满足：∫₀ˣ G(u)/u du = x/(1-x)

两边求导：G(x)/x = [x/(1-x)]' = [(1-x)-x(-1)]/(1-x)² = 1/(1-x)²

所以 G(x) = x/(1-x)²

再求导：S(x)/x = G'(x) = [x/(1-x)²]'

```
G'(x) = [(1-x)² - x·2(1-x)(-1)] / (1-x)⁴
      = [(1-x)² + 2x(1-x)] / (1-x)⁴
      = [(1-x) + 2x] / (1-x)³
      = (1+x) / (1-x)³
```

所以 S(x) = x(1+x)/(1-x)³

**第3步：代入 x = -1/2**

```
S(-1/2) = (-1/2)(1 + (-1/2)) / (1 - (-1/2))³
        = (-1/2)(1/2) / (3/2)³
        = (-1/4) / (27/8)
        = -1/4 × 8/27
        = -2/27
```

> **对照模式手册：** 题型D → 看到 ∑含n/kⁿ → 构造幂级数代入 x=-1/2

### 题31 — ∑₁^∞ (-1)ⁿ aₙ aₙ₊₂，其中 aₙ = ∫₀^∞ e^(-n²x²) dx

**解析原文跳步：** "aₙ = √π/(2n)" 和裂项求和

**完整推导：**

**第1步：计算 aₙ**

```
aₙ = ∫₀^∞ e^(-n²x²) dx
```

令 u = nx，则 du = n·dx，dx = du/n

当 x=0 → u=0；x→∞ → u→∞

```
aₙ = ∫₀^∞ e^(-u²) · (du/n)
   = (1/n) · ∫₀^∞ e^(-u²) du
```

高斯积分 ∫₀^∞ e^(-u²) du = √π/2（这是必须记住的经典结果）

所以 aₙ = √π/(2n)

**第2步：计算 aₙ·aₙ₊₂**

```
aₙ · aₙ₊₂ = [√π/(2n)] · [√π/(2(n+2))]
           = π / [4n(n+2)]
```

**第3步：裂项**

```
1/(n(n+2)) = (1/2) · [1/n - 1/(n+2)]
```

验证：(1/2)[1/n - 1/(n+2)] = (1/2)[(n+2-n)/(n(n+2))] = (1/2)[2/(n(n+2))] = 1/(n(n+2)) ✓

所以 aₙ·aₙ₊₂ = (π/4)·(1/2)·[1/n - 1/(n+2)] = (π/8)·[1/n - 1/(n+2)]

**第4步：代入级数**

```
∑₁^∞ (-1)ⁿ aₙ aₙ₊₂ = (π/8) · ∑₁^∞ (-1)ⁿ [1/n - 1/(n+2)]
```

展开前几项看规律：
```
n=1: (-1)¹[1/1 - 1/3]     = -1 + 1/3
n=2: (-1)²[1/2 - 1/4]     = +1/2 - 1/4
n=3: (-1)³[1/3 - 1/5]     = -1/3 + 1/5
n=4: (-1)⁴[1/4 - 1/6]     = +1/4 - 1/6
n=5: (-1)⁵[1/5 - 1/7]     = -1/5 + 1/7
n=6: (-1)⁶[1/6 - 1/8]     = +1/6 - 1/8
...
```

按 1/n 合并：
- 1/1：只在 n=1 出现 → -1
- 1/2：只在 n=2 出现 → +1/2
- 1/3：n=1 出现 +1/3，n=3 出现 -1/3 → 抵消
- 1/4：n=2 出现 -1/4，n=4 出现 +1/4 → 抵消
- 1/k（k≥3）：n=k-2 和 n=k 各出现一次，符号相反 → 抵消

所以只剩 -1 + 1/2 = -1/2

```
∑₁^∞ (-1)ⁿ aₙ aₙ₊₂ = (π/8) · (-1/2) = -π/16
```

> **对照模式手册：** 题型D → 积分定义 aₙ → 先算积分 → 裂项

## 题型E：傅里叶级数

### 题40 — f(x) = 1-x² 在 [0,π] 上，偶延拓后求 ∑(-1)ⁿ⁺¹/n²

**解析原文跳步：** "aₙ = 4(-1)ⁿ⁺¹/n²" 的积分过程和最终代入

**完整推导：**

**第1步：偶延拓**

将 f(x) = 1-x² 从 [0,π] 偶延拓到 [-π,π]：
```
F(x) = 1 - x²,  x ∈ [-π, π]
```

偶函数 → bₙ = 0，只有余弦级数。

**第2步：计算 a₀**

```
a₀ = (1/π) ∫₋π^π F(x) dx
   = (2/π) ∫₀^π (1-x²) dx         ← 偶函数，积分区间折半
   = (2/π) [x - x³/3]₀^π
   = (2/π) [π - π³/3]
   = 2 - 2π²/3
   = 2(1 - π²/3)
```

**第3步：计算 aₙ（n ≥ 1）**

```
aₙ = (1/π) ∫₋π^π F(x)cos(nx) dx
   = (2/π) ∫₀^π (1-x²)cos(nx) dx     ← 偶函数 × 偶函数 = 偶函数
```

拆成两部分：
```
= (2/π) [∫₀^π cos(nx) dx - ∫₀^π x²cos(nx) dx]
```

**第一个积分：**
```
∫₀^π cos(nx) dx = [sin(nx)/n]₀^π = sin(nπ)/n - 0 = 0（因为 sin(nπ) = 0）
```

**第二个积分（分部积分两次）：**
```
∫₀^π x²cos(nx) dx
```

第一次分部积分：令 u = x², dv = cos(nx)dx
```
du = 2x dx, v = sin(nx)/n

= [x²sin(nx)/n]₀^π - ∫₀^π 2x·sin(nx)/n dx
= 0 - (2/n)∫₀^π x·sin(nx) dx
```

第二次分部积分：令 u = x, dv = sin(nx)dx
```
du = dx, v = -cos(nx)/n

∫₀^π x·sin(nx) dx = [-x·cos(nx)/n]₀^π + ∫₀^π cos(nx)/n dx
                   = [-π·cos(nπ)/n + 0] + [sin(nx)/n²]₀^π
                   = -π(-1)ⁿ/n + 0
                   = -π(-1)ⁿ/n
```

代回去：
```
∫₀^π x²cos(nx) dx = -(2/n)·(-π(-1)ⁿ/n) = 2π(-1)ⁿ/n²
```

所以：
```
aₙ = (2/π) · [0 - 2π(-1)ⁿ/n²] = -4(-1)ⁿ/n² = 4(-1)ⁿ⁺¹/n²
```

**第4步：傅里叶级数**

```
S(x) = a₀/2 + ∑₁^∞ aₙcos(nx)
     = (1 - π²/3) + ∑₁^∞ [4(-1)ⁿ⁺¹/n²]·cos(nx)
```

**第5步：代入 x=0 求 ∑(-1)ⁿ⁺¹/n²**

x=0 是连续点，所以 S(0) = F(0) = 1 - 0² = 1

```
1 = (1 - π²/3) + ∑₁^∞ [4(-1)ⁿ⁺¹/n²]·cos(0)
1 = 1 - π²/3 + 4·∑₁^∞ (-1)ⁿ⁺¹/n²

π²/3 = 4·∑₁^∞ (-1)ⁿ⁺¹/n²

∑₁^∞ (-1)ⁿ⁺¹/n² = π²/12
```

> **对照模式手册：** 题型E → 构造 f(x) → 傅里叶展开 → 代入特殊点反推级数和

## 题型F：综合题

### 题10 — xₙ₊₁ = (a + xₙ)/(1 + xₙ)（0 < a < 1, x₁ > 0），证 ∑(xₙ₊₁ - xₙ) 收敛

**解析原文跳步：** 压缩不等式的推导

**完整推导：**

**第1步：算 xₙ₊₁ - xₙ**

```
xₙ₊₁ - xₙ = (a + xₙ)/(1 + xₙ) - xₙ
           = [(a + xₙ) - xₙ(1 + xₙ)] / (1 + xₙ)
           = [a + xₙ - xₙ - xₙ²] / (1 + xₙ)
           = (a - xₙ²) / (1 + xₙ)
```

**第2步：算相邻差的比（核心步骤）**

```
|xₙ₊₁ - xₙ| / |xₙ - xₙ₋₁|
```

分子：
```
|xₙ₊₁ - xₙ| = |(a + xₙ)/(1 + xₙ) - (a + xₙ₋₁)/(1 + xₙ₋₁)|
```

通分：
```
= |[(a + xₙ)(1 + xₙ₋₁) - (a + xₙ₋₁)(1 + xₙ)] / [(1 + xₙ)(1 + xₙ₋₁)]|
```

展开分子：
```
(a + xₙ)(1 + xₙ₋₁) = a + axₙ₋₁ + xₙ + xₙxₙ₋₁
(a + xₙ₋₁)(1 + xₙ) = a + axₙ + xₙ₋₁ + xₙxₙ₋₁
```

相减：
```
= (axₙ₋₁ + xₙ) - (axₙ + xₙ₋₁)
= a(xₙ₋₁ - xₙ) + (xₙ - xₙ₋₁)
= (1 - a)(xₙ - xₙ₋₁)
= (a - 1)(xₙ₋₁ - xₙ)
```

所以：
```
|xₙ₊₁ - xₙ| = |a - 1| · |xₙ₋₁ - xₙ| / [(1 + xₙ)(1 + xₙ₋₁)]
             = (1 - a) · |xₙ - xₙ₋₁| / [(1 + xₙ)(1 + xₙ₋₁)]
```

**第3步：证明 xₙ ≥ √a**

用数学归纳法。假设 xₙ ≥ 0（由 x₁ > 0 和递推保证）

```
xₙ₊₁ = (a + xₙ)/(1 + xₙ)
```

要证 xₙ₊₁ ≥ √a：
```
xₙ₊₁ - √a = (a + xₙ)/(1 + xₙ) - √a
           = [a + xₙ - √a(1 + xₙ)] / (1 + xₙ)
           = [a - √a + xₙ(1 - √a)] / (1 + xₙ)
           = [√a(√a - 1) + xₙ(1 - √a)] / (1 + xₙ)
           = (1 - √a)(xₙ - √a) / (1 + xₙ)
```

因为 0 < a < 1，所以 1 - √a > 0。如果 xₙ ≥ √a，则 xₙ₊₁ - √a ≥ 0 ✓

**第4步：得出压缩比**

因为 xₙ ≥ √a，所以 1 + xₙ ≥ 1 + √a

```
|xₙ₊₁ - xₙ| ≤ (1 - a) / (1 + √a)² · |xₙ - xₙ₋₁|
```

令 k = (1-a)/(1+√a)² < 1

等等，解析用的是 k = (1-a)/(1+a)（如果 xₙ ≥ a 而不是 √a）。让我重新看：

因为 xₙ ≥ √a > a（0 < a < 1 时 √a > a），所以 1 + xₙ ≥ 1 + a

```
|xₙ₊₁ - xₙ| ≤ (1-a)/[(1+a)(1+a)] · |xₙ - xₙ₋₁|
             ≤ (1-a)/(1+a)² · |xₙ - xₙ₋₁|
```

实际上解析用了更简单的界：因为 (1+xₙ)(1+xₙ₋₁) ≥ (1+a)²... 不，解析用的是 k = (1-a)/(1+a)

让我简化：只要证明 k < 1 就行。
```
(1-a)/(1+a) < 1 ⟺ 1-a < 1+a ⟺ -a < a ⟺ a > 0 ✓
```

**第5步：用几何级数控制**

```
|xₙ₊₁ - xₙ| ≤ k|xₙ - xₙ₋₁| ≤ k²|xₙ₋₁ - xₙ₋₂| ≤ ... ≤ kⁿ⁻¹|x₂ - x₁|
```

因为 0 < k < 1，∑kⁿ⁻¹|x₂ - x₁| 是收敛的几何级数。

由比较判别法，∑|xₙ₊₁ - xₙ| 收敛 → ∑(xₙ₊₁ - xₙ) 绝对收敛 ✓

**第6步：求级数之和**

部分和 Sₙ = ∑₁ⁿ (x_{k+1} - x_k) = xₙ₊₁ - x₁（裂项！）

设 lim xₙ = A，代入递推：
```
A = (a + A)/(1 + A)
A(1 + A) = a + A
A + A² = a + A
A² = a
A = √a
```

所以 ∑₁^∞ (xₙ₊₁ - xₙ) = lim(xₙ₊₁ - x₁) = √a - x₁

> **对照模式手册：** 题型F → 压缩映射 → 几何级数控制 → 裂项求和

## 总结：解析中最常跳的5种步骤

| 跳步类型 | 解析写法 | 实际过程 | 出现频率 |
|----------|----------|----------|----------|
| 通分化简 | "经计算得..." | 展开分子、约分、合并 | 极高 |
| 分部积分 | "积分得..." | 令u/dv、代入、可能需要两次 | 高 |
| 裂项抵消 | "∑=..." | 展开前几项，找抵消规律 | 高 |
| 求导公式 | "[f(x)]'=..." | 用商法则/链式法则，逐步展开 | 中 |
| 端点判断 | "收敛域为..." | 代入x=±R，用p级数/莱布尼茨 | 高 |

以后看到解析里任何一句"显然""易知""经计算"，都是上面这5种之一，手动展开一遍就好。
